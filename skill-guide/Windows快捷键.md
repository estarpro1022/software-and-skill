| 快捷键            | 解释                                |
| ----------------- | ----------------------------------- |
| wins + 方向键     | 靠左，靠右，全屏，最小化（不准确）  |
| wins + M/D        | 最小化所有窗口                      |
| wins + shift + M  | 将最小化窗口恢复                    |
| Alt + Esc         | 切换当前程序                        |
| Alt + F4          | 关闭当前界面或退出程序              |
| ctrl + Esc        | '开始'界面                          |
| ctrl + shift +Esc | 任务管理器                          |
| ctrl + shift + t  | delete the current browser tab      |
| win + 1/2/3       | open the application in the taskbar |
